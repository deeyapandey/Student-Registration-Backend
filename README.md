# Student Registration System – Backend API

Backend service for managing student registrations.

## Tech Stack
- ASP.NET Core
- C#
- SQL Server

## Features
- RESTful API endpoints
- Create, retrieve, update and delete student records
- Input validation
- Database persistence

## Architecture
- Layered architecture (Controllers, Services, Data)
- Clean separation of concerns
