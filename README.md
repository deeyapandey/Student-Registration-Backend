# Student Registration System – Backend API

Backend service for managing student registrations.

## Tech Stack
- ASP.NET Core
- C#
- SQL Server
- Entity Framework Core

## Features
- RESTful API endpoints for student registration
- Server-side validation
- DTOs with mapping to domain entities
- Database migrations using Entity Framework Core
- Structured error handling with appropriate HTTP status codes

## Architecture
- Layered architecture (Controllers, Services, Data)
- Separation of concerns
- Clean and maintainable code structure
- Repository pattern
